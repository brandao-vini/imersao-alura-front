# Imersão Alura: Front-End com IA 🚀

Repositório dedicado aos projetos desenvolvidos durante a **Imersão Front-End com Inteligência Artificial** da Alura. O objetivo principal é explorar como as ferramentas de IA podem otimizar o fluxo de trabalho de desenvolvimento web, desde a prototipagem até a codificação e refatoração.

---

## 🛠️ Tecnologias e Ferramentas

* **Front-End:** HTML5, CSS3, JavaScript (ES6+)
* **IA para Devs:** ChatGPT, GitHub Copilot, Gemini
* **Design:** Figma
* **Versionamento:** Git & GitHub

---

## 📝 O que foi desenvolvido

- [ ] **Aula 1** - Estruturação de página e estilização moderna.
- [ ] **Aula 2** - Integração de lógica com JavaScript.
- [ ] **Aula 3** - Refatoração de código utilizando prompts de IA.
- [ ] **Aula 4/5** - Finalização e deploy.

---

## 🚀 Como rodar o projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/imersao-frontend-ia.git](https://github.com/seu-usuario/imersao-frontend-ia.git)
    ```
2.  Abra a pasta no **VS Code**.
3.  Utilize a extensão **Live Server** para visualizar no navegador em tempo real.
